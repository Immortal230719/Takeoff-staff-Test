# This is OrcAPI SPA

based on [Create-React-App][df1]

### Tools:

1. React
2. Redux, Redux-SAGA, Redux-form
3. Material UI
4. React-router
5. Axios

### To run project locally:

```
npm i
```
and 

```
npm run start
```

#### Or, if you have Yarn

```
yarn
```

then

```
yarn start
```

 [df1]: <https://create-react-app.dev/>
