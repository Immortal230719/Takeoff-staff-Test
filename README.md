# Takeoff-Staff-Test

based on [Create-React-App][df1]

### Tools:

1. React
2. Redux, Redux-SAGA, Redux-form
3. Material UI
4. React-router
5. Axios
6. [JSON-Server][df2]

### To run project locally:

```
npm i
```
and 

```
npm run start
```

#### Or, if you have Yarn

```
yarn
```

then

```
yarn start
```

 [df1]: <https://create-react-app.dev/>
 [df2]: <https://github.com/typicode/json-server#cli-usage>
